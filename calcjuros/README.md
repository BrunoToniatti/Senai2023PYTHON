## Como Executar os Códigos

# Clone o repositório
```
git clone https://github.com/seu-usuario/pythonsenai.git
```
# Entre no diretório do projeto
```
cd pythonsenai
```
# Execute o com o Python
```
python3 calcjuros.py
```

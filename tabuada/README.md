# Tabuada em PYTHON
Criei uma tabuada em PYTHON para atividade do SENAI

# Como baixar
### Clone o repositorio:
```
git clone https://github.com/BrunoToniatti/tabuada
```
### Entre na pasta: 
```
cd tabuada
```
### Execute o python
```
python tabuada.py
```


## Como funciona
O código solicitará que você digite um número, e então ele criará um looping até que a tabuada acabe.
